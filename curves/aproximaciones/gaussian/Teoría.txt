curve_fit (SciPy) — ideas principales (para ajuste gaussiano)

* Sirve para ajustar una función a datos usando mínimos cuadrados no lineales.
* Busca parámetros que minimizan el error entre modelo y datos.

Conceptos clave:

* f(x, ...params): tu modelo (x va primero, luego parámetros).
* xdata: variable independiente (x).
* ydata: datos observados (y).
* p0: valores iniciales de los parámetros (MUY importante).
* popt: parámetros óptimos encontrados.
* pcov: covarianza estimada de los parámetros (incertidumbre aprox).
* perr = sqrt(diag(pcov)): error estándar de cada parámetro.

Muy importante en gaussianas:

* No confundir sigma de curve_fit con sigma de la gaussiana:

  * sigma (argumento de curve_fit) = incertidumbre/peso de los datos
  * sigma del modelo gaussiano = ancho de la campana

Buenas prácticas:

* Dar un buen p0 (si no, puede converger mal o fallar).
* Usar bounds para restringir parámetros (ej. amplitud > 0, sigma > 0).
* Revisar visualmente el ajuste (no confiar solo en números).
* Usar datos y modelo en float64.
* Evitar parámetros redundantes (modelo sobreparametrizado).

Métodos:

* lm: rápido, pero sin bounds.
* trf / dogbox: permiten bounds.

Alertas:

* Si parámetros tienen escalas muy distintas, el ajuste puede fallar.
* pcov grande o mal condicionada => resultados poco confiables.
* curve_fit hace optimización local (puede caer en mínimo local).

En resumen:
* Modelo correcto + buen p0 + bounds + revisar popt/pcov = ajuste mucho más confiable.

Aproximación de curvas con método gaussiano — ideas principales

* Es una técnica para ajustar una curva gaussiana (forma de campana) a un conjunto de datos.
* Se usa cuando los datos presentan uno o más picos.

Idea principal:

* Encontrar los parámetros de la gaussiana que mejor se ajustan a los datos minimizando el error.

Modelo gaussiano:

* y = A * exp(-((x - mu)^2) / (2*sigma^2)) + c

Parámetros:

* A: amplitud (altura)
* mu: media / centro del pico
* sigma: ancho o dispersión
* c: offset o línea base

Conceptos clave:

* Residual = y_observado - y_modelo
* Se minimiza la suma de residuales al cuadrado (mínimos cuadrados)
* El ajuste es una aproximación

Cuándo funciona mejor:

* Cuando los datos tienen forma aproximadamente gaussiana
* Cuando hay poco ruido y pocos valores atípicos (outliers)
* Cuando se eligen buenos valores iniciales

Aplicaciones:

* Espectros (Raman, IR, masas)
* Señales experimentales
* Estadística y distribuciones
* Procesamiento de imágenes/datos

Ventajas:

* Simple, útil e interpretable
* Pocos parámetros con significado claro

Limitaciones:

* No ajusta bien curvas no gaussianas
* Puede fallar con mucho ruido o malos valores iniciales
* Si hay varios picos, puede requerir suma de gaussianas

Resumen:

* Aproximación gaussiana = representar datos con una campana (o varias) para describir su forma de manera simple.
